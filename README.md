# Clínica Podológica Julia - Landing Page Premium

Landing page de alto impacto diseñada para la **Clínica Podológica Julia**, con una estética "Dark Luxury" y acento en **Verde Neón**.

## 🚀 Características
- **Diseño Dark Mode Premium**: Fondos negros profundos y elementos con efecto vidrio (glassmorphism).
- **Interactividad**: Flip cards 3D para servicios, sistema de partículas de fondo y animaciones de scroll.
- **Responsivo**: Adaptado perfectamente para móviles, tablets y escritorio.
- **Optimizado para Salud**: Estructura orientada a la conversión y confianza del paciente.

## 🎨 Sistema de Diseño
- **Color Primario**: Verde Neón (`#22C55E`)
- **Tipografía**:
  - Títulos: Plus Jakarta Sans
  - Cuerpo: Inter
- **Efectos**: Sombras de neón (glow), desenfoque de fondo y transiciones suaves.

## 📂 Estructura del Proyecto
- `index.html`: Estructura semántica con SEO optimizado.
- `styles.css`: Estilos modernos utilizando variables CSS.
- `script.js`: Efectos dinámicos y lógica de navegación.
- `/images/`: Directorio para recursos visuales.

## 📸 Notas sobre las imágenes
Debido a una alta demanda momentánea en el servicio de generación de imágenes, se han utilizado URLs de imágenes profesionales de stock y SVGs de alta calidad para garantizar la entrega inmediata. Puedes reemplazar estas imágenes en el `index.html` por fotos propias de la clínica.

## 🛠️ Personalización
Para cambiar el color de acento, simplemente modifica la variable `--primary-500` en el archivo `styles.css`.
